# Build Instructions

## 1. Install environment
- C++ Build tool CMAKE - https://cmake.org/
- Vulkan SDK - https://vulkan.lunarg.com/

## 2. Clone repository and dependencies
```
git clone https://github.com/CompilerLuke/eth_localization.git --recursive
```

## 3. Build using cmake
```
cd eth_localization
mkdir build
cd build
cmake ../
make -j 8
```
